# 图片
## img标签
img标签中，src和alt是必须要填的
src:表示图片的路径
alt：表示图片找不到的时候，要显示的内容
代码如下：

![img.png](img.png)

效果如下：

![img_1.png](img_1.png)

当图片地址不存在时，即找不到图片时，显示效果如下：

![img_2.png](img_2.png)

显示小图片裂开状态，并显示alt的内容。
